# Project Layout

## Top-Level Folders

- `docs/`: architecture, demo narrative, and hackathon framing for judges and collaborators.
- `frontend/`: React + Vite + TypeScript mockup-first product console for the Compass demo.
- `services/crti-api/`: FastAPI mock service that supplies the frontend with hackathon-ready JSON endpoints.
- `canton/`: lightweight DAML package skeleton showing the intended Canton workflow and selective visibility model.
- `quickstart/`: reserved area for the underlying Canton runtime and upstream quickstart stack.

## Key Files

- `README.md`: public project overview and run instructions.
- `LICENSE`: MIT license for the hackathon repository.
- `.gitignore`: ignores frontend, Python, DAML, IDE, and environment artifacts.
