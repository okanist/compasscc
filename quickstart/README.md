# Quickstart Placeholder

This folder is reserved for the underlying Canton runtime, local environment setup, or upstream quickstart stack that would back a fuller Compass deployment.

The public hackathon repository does not attempt to bundle the full official Canton quickstart automatically. Instead, this directory marks the intended integration point for:

- Canton node configuration
- participant and domain runtime assets
- local developer bootstrap scripts
- upstream stack references used in a fuller implementation

For the current hackathon submission, the frontend, mock API, docs, and DAML package skeleton are the primary demo assets.
